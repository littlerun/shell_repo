+----------------------------------+
 Xweibo 数据库还原工具
+----------------------------------+
本工具用于无法在后台恢复数据库的时候进行数据恢复操作。

+----------------------------------+
 操作步骤
+----------------------------------+
1、将[安装包]\tools\db_restore\upload下面的所有文件上传到[xweibo安装目录]下
2、访问http://[xweibo地址]/restore/index.php
3、按提示操作
4、删除[xweibo安装目录]/restore
